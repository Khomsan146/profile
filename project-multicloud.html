The Challenge

The organization required scalable compute power for peak loads but needed to keep sensitive customer data within their local data center due to strict compliance laws. A pure cloud or pure on-prem solution was not viable.

The Solution

Implemented a Hybrid Multi-Cloud architecture connecting on-prem infrastructure to AWS via Site-to-Site VPN.

  • Network Integration: Established redundant VPN tunnels between AWS VPC and the local Data Center Firewall to ensure secure, private connectivity.
  • Workload Distribution: Frontend and stateless services were deployed on AWS EKS to handle traffic spikes, while databases remained on-prem on high-performance Nutanix clusters.
  • Unified Management: Used Rancher/ArgoCD to manage Kubernetes clusters across both environments from a single pane of glass.
  • Disaster Recovery: Configured AWS S3 for offsite backups and implemented a warm standby DR plan in a secondary AWS region.
